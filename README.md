
# G_Bus 🚌

G_Bus is a Java Console-Based Bus Ticket Booking System that simulates a real-world bus reservation platform. The application provides an interactive terminal experience with multithreading, ASCII animations, seat booking, ticket management, and admin controls.

## ✨ Features

### 👤 User Features
- User Registration
- User Login
- View Available Buses
- Search Buses
- Book Tickets
- Cancel Tickets
- View Booking History

### 🔑 Admin Features
- Admin Login
- Add New Buses
- Remove Buses
- View All Bookings
- Revenue Tracking

### 🎨 Interactive Console UI
- ASCII Welcome Screen
- Bus Search Animation
- Payment Processing Animation
- Ticket Printing Animation
- Seat Layout Visualization

### 🧵 Multithreading
- Startup Loading Animation
- Payment Processing Spinner
- Ticket Printing Effects

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Collections Framework
- Multithreading
- Exception Handling
- Console UI Design
- ASCII Art

## 🚀 Getting Started

### Prerequisites

* Java JDK 8 or higher
* Windows Operating System

### Run the Application

Simply execute:

```cmd
run.bat
```

The batch file will automatically compile and launch the application.
